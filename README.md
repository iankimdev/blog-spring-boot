# blog

유저 인증기능과 포스팅 기능을 갖춘 블로그

- Java와 springboot를 공부하기 위한 토이 프로젝트입니다.
- todo app을 만들며 MVC패턴과 CRUD를 이해하였습니다.

## 기능
- Web Security와 BCryptPasswordEncoder을 활용한 유저 인증 기능
- Create new post 버튼으로 포스팅 추가
- Update 버튼으로 포스팅 수정
- Delete 버튼으로 포스팅 삭제
